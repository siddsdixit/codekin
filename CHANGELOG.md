# Changelog

## [1.0.0] - 2025-11-18

### Added
- Initial release of Codekin
- Multi-agent AI coding system with 6 specialized agents
- Smart orchestration with parallel execution
- Database persistence with SQLite
- Code indexing with Qdrant
- Support for Anthropic Claude, OpenAI GPT, and other providers

### Features
- **6 Specialized Agents**: PM, Architect, Frontend Dev, Backend Dev, QA, DevOps
- **Parallel Execution**: Independent tasks run simultaneously for 2-3x speedup
- **Tool Restrictions**: Each agent limited to specific tools
- **File Restrictions**: Glob pattern matching prevents unauthorized file access
- **Database Persistence**: Full conversation history and task tracking
- **Code Indexing**: Semantic search with embeddings
- **Configuration System**: Environment variables and config file support

### Based On
- Forked from Roo Code
- Built on proven foundation with 22+ tools and 40+ AI providers
